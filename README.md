# Passengers app

- Simple app built with React, Bootstrap and JSON-server for API.
- Application built as a simple and basic React application, with very basic functionality.

# Approach

- to comply with simplistic requirement all code is simplified: components are not split to extra component whenever possible, and there is number of hardcoded values, like API url etc.
- I followed presentational/container components with three containers: `Passengers`, `PassengersEditor` and `PassengersCreator` and everything else being presentational components.
- some functions could be extracted to other files, but they are left out for simplicity purposes.
